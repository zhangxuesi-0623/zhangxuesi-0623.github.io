# 个人学术主页

这是一个基于 GitHub Pages 的个人学术主页项目，展示医学影像学专业学生张学思的个人信息和学术成果。

## 功能特点

- ✅ 响应式设计，支持移动端
- ✅ 动态导航条
- ✅ 平滑滚动效果
- ✅ 交互式医学影像知识图谱
- ✅ 时间线展示教育经历
- ✅ 卡片式布局展示经历和获奖

## 技术栈

- HTML5
- CSS3 (Grid, Flexbox, 动画)
- JavaScript (ES6+)
- Font Awesome 图标

## 部署到 GitHub Pages

1. 在 GitHub 创建新仓库：`username.github.io` (username 替换为你的用户名)
2. 将所有文件上传到仓库
3. 在仓库设置中启用 GitHub Pages
4. 访问 `https://username.github.io` 查看网站

## 本地运行

直接在浏览器中打开 `index.html` 文件，或使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
